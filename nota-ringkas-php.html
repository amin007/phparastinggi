Bab Pengenalan : Objektif

Objektif modul panduan ini adalah memberi panduan kepada para pembangun laman web yang ingin mengetahui tentang asas-asas utama bahasa pengaturcaraan Php dan Mysql

Modul panduan ini merangkumi:

  1. Pengenalan ringkas Apache, Php dan Mysql.
  2. Membina borang untuk $_POST, $_FILES dan $_GET.
  3. Memahami tatasusunan.
  4. Apakah yang dikatakan fungsi.
  5. Berkenalan dengan class.
  6. Cara membina hubungan antara Php & Mysql.

Bab Pengenalan : Keperluan

Untuk mengikuti panduan ini. Anda perlu mempunyai asas dan pengalaman dalam pembangunan web seperti :

  • Tahu mengunakan google aka tahu keyword yang sesuai.
  • Mempunyai asas HTML, CSS, dan Javascript.
  • Tahu mereka bentuk struktur dan skema pengkalan data (Databases)
  • Tahu menggunakan alat pengurusan pengkalan data(Database Administration Tools)
    seperti PhpMyAdmin, Admirer, Sequel Pro, dll.

Bab 2a : Definasi $_POST, $_FILES dan $_GET.
  • Ia dipanggil => Pembolehubah Pelayan Server
  • Pembolehubah pelayan server disimpan dalam bentuk tatasusunan (array).
  • Untuk melihat nilai-nilai yang disimpan dalam tatasusunan ini, gunakan fungsi print_r() atau foreach().
Bab 2a : Definasi Pembolehubah Pelayan Server.
KodKeterangan
$_POSTpembolehubah yang ada pada script dan dihantar melalui kaedah HTTP POST
$_GETpembolehubah yang ada pada script dan dihantar melalui kaedah HTTP GET
$_FILESpembolehubah yang ada pada script dan dihantar melalui kaedah HTTP POST muatnaik (upload file)
Bab 2a : Definasi Pembolehubah Pelayan Server.
KodKeterangan
$_SERVERpembolehubah yang telah ditetapkan oleh pelayan server
$_SESSIONpembolehubah yang terdapat pada sesi semasa yang telah ditetapkan. Sesi ini disimpan dalam pelayan server.
$_COOKIEpembolehubah yang terdapat pada sesi semasa yang telah ditetapkan. Namun begitu, sesi ini disimpan dalam komputer pengguna.

Bab 2b : Keterangan maksud $_FILES.
Senarai Keterangan
$_FILES[***]['name'] Nama asal fail sebelum muat naik
$_FILES[***]['type'] Jenis-jenis fail aka MIME Type
$_FILES[***]['size'] Saiz fail (dalam kiraan Bytes)
$_FILES[***]['tmp_name'] Lokasi sementara fail dalam server
$_FILES[***]['error'] Sebarang kod amaran semasa muatnaik fail
Bab 2b : Latihan borang yang mengunakan $_FILES.

Bab 2b : Keterangan maksud $_FILES[***]['name'].

Periksa sama ada fail Sudah Wujud. Sekarang kita boleh menambah beberapa sekatan.
Pertama, kita akan memeriksa jika fail yang telah wujud dalam "muat naik" folder. Jika ya, mesej ralat akan dipaparkan, dan $uploadOk ditetapkan kepada 0: 

<?php // Check if file already exists
if (file_exists($target_file)) 
{
    echo "Sorry, file already exists.";
    $uploadOk = 0;
}
?>

Bab 2b : Keterangan maksud $_FILES[***]['size'].

Bidang input fail dalam bentuk HTML kami di atas dinamakan "fileToUpload". Sekarang, kita mahu untuk memeriksa saiz fail. Jika fail yang lebih besar daripada 500KB, mesej ralat akan dipaparkan, dan $uploadOk ditetapkan kepada 0:

<?php // Check file size
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Sorry, your file is too large.";
    $uploadOk = 0;
}
?>
Bab 2b : Keterangan maksud $_FILES[***]['type'].

Kod di bawah sahaja membolehkan pengguna untuk memuat naik JPG, JPEG, PNG, dan fail GIF. Semua jenis fail lain memberikan mesej ralat sebelum menetapkan $uploadOk ke 0:

<?php // Allow certain file formats
if($imageFileType != "jpg" && $imageFileType != "png" && $imageFileType != "jpeg"
&& $imageFileType != "gif" ) {
    echo "Sorry, only JPG, JPEG, PNG & GIF files are allowed.";
    $uploadOk = 0;
}
?>
Bab 2b : Papar semua koding

Fail "upload.php" yang lengkap kelihatan seperti ini:

<?php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$imageFileType = pathinfo($target_file,PATHINFO_EXTENSION);
// Check if image file is a actual image or fake image
if(isset($_POST["submit"])) {
    $check = getimagesize($_FILES["fileToUpload"]["tmp_name"]);
    if($check !== false) {
        echo "File is an image - " . $check["mime"] . ".";
        $uploadOk = 1;
    } else {
        echo "File is not an image.";
        $uploadOk = 0;
    }
}
// Check if file already exists
if (file_exists($target_file)) {
    echo "Sorry, file already exists.";
    $uploadOk = 0;
}
// Check file size
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Sorry, your file is too large.";
    $uploadOk = 0;
}
// Allow certain file formats
if($imageFileType != "jpg" && $imageFileType != "png" && $imageFileType != "jpeg"
&& $imageFileType != "gif" ) {
    echo "Sorry, only JPG, JPEG, PNG & GIF files are allowed.";
    $uploadOk = 0;
}
// Check if $uploadOk is set to 0 by an error
if ($uploadOk == 0) {
    echo "Sorry, your file was not uploaded.";
// if everything is ok, try to upload file
} else {
    if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
        echo "The file ". basename( $_FILES["fileToUpload"]["name"]). " has been uploaded.";
    } else {
        echo "Sorry, there was an error uploading your file.";
    }
}
?>
