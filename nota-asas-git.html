Apa / Mengapa / Bila

  • Apa itu Git
  • Mengapa gunakan Git
  • Bila masa gunakan Git

Apa itu Git

Git adalah sumber terbuka sistem kawalan versi diedarkan (Dvcs) yang ketara boleh meningkatkan cara anda bekerja dan bekerjasama dalam projek.

Mengapa gunakan Git

Git membolehkan anda menyimpan sejarah perubahan penting yang dibuat kepada fail tempatan anda.

Ia juga boleh membuat sandaran sejarah ini melalui perkhidmatan hosting terpencil seperti GitHub.

Bila masa gunakan Git

Membantu anda bekerja bersendirian atau dengan pasukan.
